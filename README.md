# Healthcare Appointment Attendance Analytics

## Project Overview

This project analyzes 110,000+ healthcare appointments to identify factors contributing to patient no-shows and provide actionable recommendations to improve appointment attendance.

## Tech Stack

- PostgreSQL
- SQL
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook
- Power BI
- Git & GitHub
